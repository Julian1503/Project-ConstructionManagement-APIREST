# Project Construction Management Backend
## Description
This project is the backend part of Project-ConstructionManagement.
With this project you can do CRUD to the data base.

## Technologies used:
* C#.
* Code first.
* T-SQL.
* API REST.
* Swagger.
* ASP .NET CORE.
* .NET CORE.
* AutoMapper.
* MVC.
